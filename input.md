# Input

See [docs](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/input)

## color

A control for specifying a color; opening a color picker when active in supporting browsers.

```html
<input type="color" name="color" />
```

## date

A control for entering a date (year, month, and day, with no time). Opens a date picker or numeric wheels for year, month, day when active in supporting browsers.

```html
<input type="date" name="date" />
```

## datetime-local

A control for entering a date and time, with no time zone. Opens a date picker or numeric wheels for date- and time-components when active in supporting browsers.

```html
<input type="datetime-local" name="datetime-local" />
```

## email

```html
<input type="email" name="email" />
```

## image

A graphical submit button. Displays an image defined by the src attribute.

```html
<input
  id="image"
  type="image"
  width="100"
  height="30"
  alt="Login"
  src="https://raw.githubusercontent.com/mdn/learning-area/master/html/forms/image-type-example/login.png" />
```

## month

A control for entering a month and year, with no time zone.

```html
<input type="month" name="month" />
```

## number

```html
<input type="number" name="score" min="1" max="5" />
```

## range

```html
<input type="range" name="range" min="0" max="25" />
```

## search

A single-line text field for entering search strings. Line-breaks are automatically removed from the input value. 
May include a delete icon in supporting browsers that can be used to clear the field. 
Displays a search icon instead of enter key on some devices with dynamic keypads.

```html
<input type="search" name="search" />
```

## tel

A control for entering a telephone number. Displays a telephone keypad in some devices with dynamic keypads.

```html
<input type="tel" name="tel" />
```

## time

A control for entering a time value with no time zone.

```html
<input type="time" name="time" />
```

## url

A field for entering a URL. Looks like a text input, but has validation parameters and relevant keyboard in supporting browsers and devices with dynamic keyboards.

```html
<input type="url" name="url" />
```

## week

A field for entering a URL. Looks like a text input, but has validation parameters and relevant keyboard in supporting browsers and devices with dynamic keyboards.

```html
<input type="week" name="week" />
```
